Shotumi

« Shotumi, un shooter de l'espace aux couleurs océanique. »
Il s'agit d'un jeu vidéo de type Space Shooter où le principe est de détruire tous les astéroïdes qui arrivent sur soi avant qu'ils ne vous détruisent. Vous possedez un nombre de vie limitée et vous l'aurez compris vous devez survivre le plus longtemps possible dans cette eau spatiale (pas en apnée attention).

À propos du jeu

Shotumi est jeu qui a été developpé en langage Python lors de ma formation Game Developer à Interface3.
J'ai choisi un thème relié à l'océan afin de pouvoir y intégrer un requin baleine, un poisson marin qui me fascine. Pour le vaisseau requin baleine je me suis inspirée notamment d'Abzû, un jeu vidéo du studio Giant Squid où l'on nage dans l'océan ainsi que des tatouages polynésiens pour les dessins sur le vaisseau.
